## 🎯 欢迎来到 NULS AI 文档中心

### 什么是 NULS AI？

NULS AI 是一个提供可定制服务的区块链基础设施和全球开源社区区块链项目。我们采用微服务实现高度模块化的底层架构，使用智能合约和跨链技术，结合 ChainBox 快速建链的能力，降低开发成本，加速区块链商业应用落地。

## 🌟 核心特色

### 🧩 模块化架构
NULS AI 采用微服务架构，每个功能模块都可以独立开发、部署和升级，提供高度的灵活性和可扩展性。开发者可以根据需求选择和组合不同的模块。

### 🔗 跨链技术
支持多链资产转移和交互，实现不同区块链网络之间的价值流通和信息传递。通过先进的跨链协议，用户可以无缝地在不同链之间转移资产。

### 🤖 AI 集成
集成AI Agent等功能，让区块链更加智能和易用。

### ⚡ 快速建链
通过 ChainBox 工具，开发者可以快速构建和部署自定义区块链，大大降低开发门槛，让区块链技术更加普及。
